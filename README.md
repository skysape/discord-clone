# Discord Clone (простой демо-мессенджер)

Простое веб-приложение на Java (Spring Boot) с функциями:

- Регистрация / вход
- Добавление друзей (заявки, принятие/отклонение)
- Создание групп и добавление в них друзей
- Никнейм и аватарка (загрузка фото из галереи)
- Голосовые звонки 1-на-1 через WebRTC:
  - ПКМ по другу во время звонка — регулировка его громкости
  - Кнопка «заглушить микрофон» (mute)
  - Кнопка «выключить звук» (deafen)
- Интерфейс на 4 языках: русский, беларуская, polski, English

## Требования

- Java 17+
- Maven 3.8+ (или используйте `./mvnw`, если добавите wrapper)
- Современный браузер (Chrome/Firefox) — для доступа к микрофону нужен HTTPS
  либо `localhost` (на localhost микрофон работает и без HTTPS).

## Запуск

```bash
mvn spring-boot:run
```

После запуска откройте в браузере:

```
http://localhost:8080
```

Приложение хранит данные в файловой базе H2 (`./data/discorddb.mv.db`),
она создастся автоматически при первом запуске.

Загруженные аватарки сохраняются в папку `./uploads/avatars`.

## Как потестировать звонок

Голосовой звонок — это WebRTC-соединение точка-точка. Чтобы проверить его
локально, откройте приложение в двух разных вкладках/браузерах, зарегистрируйте
двух разных пользователей, добавьте их друг друга в друзья и нажмите «Позвонить».

## Важно

Это учебный/демонстрационный проект:

- Токены авторизации хранятся в памяти сервера (при перезапуске сервера
  все пользователи разлогинятся).
- Нет rate-limiting, восстановления пароля, шифрования сообщений и т.п.
- WebRTC использует только публичный STUN-сервер Google; если у собеседников
  «сложная» сеть (симметричный NAT), может понадобиться TURN-сервер.

Дальше проект можно расширять: текстовые сообщения в группах, статусы
«онлайн/оффлайн», групповые звонки, push-уведомления и т.д.
